Getting back into markdown, here's my quick lazy cheatsheet.


\_ \_ : italics
\** \** : bold
\[ \] (link) : hyperlink
\- : bullet point
\![alt](imagelink) : image
\<details id=1> </details> : collapsable
\<summary></summary> : title for collapsed
\> : quote
\<!-- \--> : 주석
\:sparkles: : emoji
\` \` : short code
\``` yml \``` : yml code
